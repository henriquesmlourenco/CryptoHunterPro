PORTUGUES:
  Qual usar?
  - Se o Windows for 64-bit (maioria): crypto_hunter_pro_x64.exe
  - Se o Windows for 32-bit (antigo):  crypto_hunter_pro_x86.exe
  - Na duvida, use o x64. Se nao abrir, tente o x86.

  NOTA: O Windows Defender/SmartScreen pode alertar na primeira
  execucao por ser um programa nao-assinado. Isso e normal.
  Clique em "Mais informacoes" e depois "Executar mesmo assim".


ENGLISH:
  Which one should you use?
  - If your Windows is 64-bit, which is the most common: crypto_hunter_pro_x64.exe
  - If your Windows is 32-bit, usually on older PCs: crypto_hunter_pro_x86.exe
  - If you are not sure, try the x64 version first. If it does not open, use the x86 version.

  Note:
  Windows Defender or SmartScreen may show an alert the first time you run it because the program is unsigned. This is normal.
  Click “More info” and then “Run anyway.”
